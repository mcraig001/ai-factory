## DESIGN PRINCIPLES (TensorPoet Framework)

### Visual Quality Standards
1. **No AI Slop**:
   - ❌ Purple gradients
   - ❌ Lucide icons (overused)
   - ❌ Generic layouts
   - ✅ Clean, professional, corporate aesthetic

2. **Typography**:
   - Headers: Inter Bold or Newsreader
   - Body: Inter Regular
   - Data: Fira Code (monospace for numbers)
   - Avoid: Default Excel fonts

3. **Color Palette** (Professional Finance):
   - Primary: #1e3a8a (Deep Blue)
   - Secondary: #3b82f6 (Bright Blue)
   - Success: #10b981 (Green for positive)
   - Warning: #f59e0b (Orange for caution)
   - Danger: #ef4444 (Red for negative)
   - Neutral: #6b7280 (Gray for text)
   - Background: #ffffff, #f9fafb

4. **Layout**:
   - Grid-based structure
   - Clear visual hierarchy
   - Generous white space
   - Professional, not playful

## PRODUCTION WORKFLOW

### Phase 1: Research (30 minutes per template)
1. Check Gumroad, Creative Market, Envato for similar templates
2. Analyze top sellers (what makes them successful?)
3. Identify gaps (what's missing in market?)
4. Read reviews (what do customers want?)
5. Determine pricing (based on market + quality)

### Phase 2: Design (2-3 hours per template)
1. Create structure in Excel/Sheets/Notion
2. Build formulas and calculations
3. Apply design system (colors, fonts, layouts)
4. Add example data
5. Create 2-3 variations (different styles/industries)

### Phase 3: Quality Control (30 minutes)
1. Test all formulas
2. Verify calculations
3. Check for errors
4. Score against quality checklist
5. If <85, iterate until >85

### Phase 4: Packaging (30 minutes)
1. Create README/Instructions document
2. Add screenshots for marketplace
3. Write compelling description
4. Set pricing
5. Package files (ZIP)

### Phase 5: Submit for Approval
1. Upload to CEO Dashboard
2. Include: Preview, quality score, pricing rationale
3. Wait for approval
4. If approved → Launch
5. If rejected → Iterate based on feedback
